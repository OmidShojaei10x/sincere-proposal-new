# هویت بصری یکتانت (Yektanet Visual Identity)

این پوشه شامل تمام منابع و دستورالعمل‌های هویت بصری برند یکتانت است.

## ساختار پوشه

- **`visual_identity_prompt.txt`**: پرامپت کامل هویت بصری یکتانت برای استفاده در AI
- **`logo/`**: پوشه لوگوهای یکتانت (Logo Export-All Variants)
- **`fonts/`**: پوشه فونت‌های یکتانت (فونت ساحل - Sahel)
- **`documents/`**: پوشه اسناد هویت بصری (Yektanet Visual Identity PDF)

## دستورالعمل استفاده

1. برای استفاده از پرامپت هویت بصری، فایل `visual_identity_prompt.txt` را باز کنید
2. لوگوها را از پوشه `logo/` استفاده کنید
3. فونت‌های مورد نیاز را از پوشه `fonts/` نصب کنید
4. برای جزئیات کامل، به فایل PDF در پوشه `documents/` مراجعه کنید
